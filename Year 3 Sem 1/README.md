This directory contains course folders for Year 3 Semester 1. Click a course to open its folder.

- **[Database Programming](Database Programming/):** Advanced database programming, stored procedures, and database-driven application development.
- **[Enterprise System Design and Modelling (WBL)](Enterprise System Design and Modelling (WBL)/):** Enterprise modelling techniques applied in workplace-based learning.
- **[Field Astronomy](Field Astronomy/):** Practical astronomy studies including observations and celestial measurements.
- **[French Language](French Language/):** Basic French language skills for communication and cultural understanding.
- **[Professional Communication Skill 2](Professional Communication Skill 2/):** Advanced professional communication topics and presentations.
