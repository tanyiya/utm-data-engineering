# Year 2 — Semester 2

This directory contains course folders for Year 2 Semester 2. Click a course to open its folder.

- **[Data Analytics Programming](Data Analytics Programming/):** Programming techniques and tools for data analysis and scripting.
- **[Data Mining](Data Mining/):** Methods for extracting patterns and knowledge from large datasets.
- **[Object Oriented Programming](Object Oriented Programming/):** Object-oriented design, classes, inheritance, and polymorphism in modern languages.
- **[Operating System](Operating System/):** Concepts of operating systems, process management, memory, and I/O.
- **[Philosophy and Current Issue](Philosophy and Current Issue/):** Examination of philosophical ideas and contemporary societal issues.
- **[Professional Communication Skill](Professional Communication Skill/):** Developing written and oral communication skills for professional contexts.
- **[Technopreneurship Seminar](Technopreneurship Seminar/):** Entrepreneurship in technology, startup concepts, and innovation practices.
