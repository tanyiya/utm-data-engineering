# Year 1 — Semester 2

**Expanding into hardware architecture, human-centred design, statistics, and systems thinking.**

</div>

---

Building on the foundations of Semester 1, this semester introduces computer architecture, user interface design, probability and statistics, and applied systems analysis — broadening both the technical and analytical dimensions of the programme.

---

## Courses

### 🏛️ [Appreciation of Ethics and Civilizations](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%201%20Sem%202/Appreciation%20of%20Ethics%20and%20Civilizations)
**Study of ethical frameworks and civilisational development.**

Examines major ethical traditions and civilisational history, fostering critical reflection on cultural values and the responsibilities that accompany technological advancement.

---

### ⚙️ [Computer Organisation and Architecture](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%201%20Sem%202/Computer%20Organisation%20and%20Architecture)
**Internal architecture of computer systems and assembly language.**

Covers CPU design, instruction set architecture, memory hierarchy, pipelining, and I/O organisation. Practical assembly language labs reinforce the relationship between high-level code and hardware execution.

---

### 🖱️ [Human Computer Interaction](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%201%20Sem%202/Human%20Computer%20Interaction)
**User-centred design principles and interface evaluation.**

Introduces usability principles, interface design methodologies, prototyping techniques, and evaluation methods including heuristic analysis and user testing. Grounds technical development in the needs and behaviours of real users.

---

### 📊 [Probability & Statistical Data Analysis](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%201%20Sem%202/Probability%20&%20Statistical%20Data%20Analysis)
**Probability theory and statistical methods for data analysis.**

Covers probability distributions, descriptive statistics, hypothesis testing, confidence intervals, and regression analysis. Provides the quantitative foundation for data analytics and machine learning coursework in later years.

---

### 💻 [Programming Technique II](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%201%20Sem%202/Programming%20Technique%20II)
**Intermediate programming and structured software development in C/C++.**

Advances core programming skills through pointers, dynamic memory management, file I/O, and structured project development. Strengthens the ability to design and implement well-organised programs.

---

### 📋 [System Analysis and Design (WBL)](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%201%20Sem%202/System%20Analysis%20and%20Design%20(WBL))
**Systems analysis methodologies applied in a workplace-based learning context.**

Covers requirements elicitation, process modelling, data flow diagrams, entity-relationship diagrams, and system design approaches — applied to real organisational scenarios through a structured work-based learning placement.
