# Year 2 — Semester 1

**Core computer science: algorithms, databases, networks, and software engineering.**

</div>

---

Semester 1 of Year 2 represents a significant increase in technical depth. Coursework covers the fundamental pillars of applied computing — algorithm design, relational databases, network communication, and the software engineering discipline — alongside practical system development tools.

---

## Courses

### 🌾 [Agricultural Skills](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%202%20Sem%201/Agricultural%20Skills)
**Community service learning in agricultural practice.**

A compulsory experiential learning component providing practical exposure to sustainable agricultural methods and rural operations. Develops adaptability, teamwork, and an appreciation for domains beyond computing.

---

### 🌲 [Data Structure and Algorithm](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%202%20Sem%201/Data%20Structure%20and%20Algorithm)
**Algorithm design, analysis, and fundamental data structures.**

Studies arrays, linked lists, stacks, queues, trees, heaps, hash tables, and graphs alongside sorting and searching algorithms. Emphasises time and space complexity analysis as the basis for writing efficient, scalable code.

---

### 🗄️ [Database](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%202%20Sem%201/Database)
**Relational database design and SQL.**

Covers the relational model, entity-relationship modelling, normalisation through BCNF, and comprehensive SQL — including queries, joins, subqueries, views, and transaction management. Forms the basis for advanced database coursework in Year 3.

---

### 🌐 [Network Communication](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%202%20Sem%201/Network%20Communication)
**Network architecture, protocols, and communication fundamentals.**

Examines the OSI and TCP/IP models, IP addressing, subnetting, routing and switching protocols, and network configuration. Practical lab components use network simulation tools to reinforce theoretical concepts.

---

### 🛠️ [Software Engineering](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%202%20Sem%201/Software%20Engineering)
**Software development methodologies, modelling, and project management.**

Covers the full software development lifecycle, Agile and Waterfall methodologies, UML modelling, design patterns, software testing strategies, and team-based project delivery practices.

---

### 🚀 [System Development Technology](https://github.com/tanyiya/utm-data-engineering/blob/main/Year%202%20Sem%201/Sytem%20Development%20Technology)
**Tools, platforms, and technologies for modern system development.**

Introduces version control with Git, development environment configuration, deployment pipelines, and the technology stack used in professional software development and system delivery workflows.
