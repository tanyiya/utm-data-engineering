# Year 2 — Semester 1

This directory contains course folders for Year 2 Semester 1. Click a course to open its folder.

- **[Agricultural Skills](Agricultural Skills/):** Practical skills and knowledge related to agriculture and farm practices.
- **[Data Structure and Algorithm](Data Structure and Algorithm/):** Study of data structures, algorithm design, complexity, and implementation in code.
- **[Database](Database/):** Fundamentals of database design, SQL, and relational database management systems.
- **[Network Communication](Network Communication/):** Basics of networking, protocols, and communication models.
- **[Software Engineering](Software Engineering/):** Software development lifecycle, modelling, design patterns, and project management.
- **[Sytem Development Technology](Sytem Development Technology/):** Tools and technologies used in system development and deployment.
