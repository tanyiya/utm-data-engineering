# Year 2 — Semester 1

This directory contains course folders for Year 2 Semester 1. Click a course to open its folder.

- **[Agricultural Skills](Agricultural%20Skills/):** Practical skills and knowledge related to agriculture and farm practices.
- **[Data Structure and Algorithm](Data%20Structure%20and%20Algorithm/):** Study of data structures, algorithm design, complexity, and implementation in code.
- **[Database](Database/):** Fundamentals of database design, SQL, and relational database management systems.
- **[Network Communication](Network%20Communication/):** Basics of networking, protocols, and communication models.
- **[Software Engineering](Software%20Engineering/):** Software development lifecycle, modelling, design patterns, and project management.
- **[Sytem Development Technology](Sytem%20Development%20Technology/):** Tools and technologies used in system development and deployment.
